# CarCulator - Comparador de Compra x Aluguel

## Sobre o projeto

Criação de um site que auxilia nos cálculos comparativos entre:

 - Comprar um carro à vista
 - Comprar um carro financiado
 - Alugar um carro

 O sistema é dividido em:
 - Backend: Node.js + Typescript + Express
 - Frontend: React + Vite

---

## Instalação de dependêncis

### Pré-requisitos

- Node.js
- npm ou yarn

---

### 1. Instalar dependências do backend

Na raiz do projeto:

```bash
npm install
```

### 2. Instalar dependências do frontend

```bash
cd frontend/CarCulator
npm install
```

## Como executar

### Modo desenvolvimento

#### Backend

```bash
npm run dev
```

#### Frontend

```bash
cd frontend/CarCulator
npm run dev
```

---

### Build para produção

#### Backend
```bash
cd backend
npx tsc
npm start
```

#### Frontend

```bash
cd frontend/CarCulator
npm run build
```
